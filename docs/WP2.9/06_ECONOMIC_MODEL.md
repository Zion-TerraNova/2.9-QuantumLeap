### Bonus Distribution Logic

```python
def calculate_consciousness_reward(block_number, miner_level):
    """
    Calculate consciousness bonus for a block
    Only active 2025-2035 (first 10 years)
    """
    CONSCIOUSNESS_POOL = 8_250_000_000
    DISTRIBUTION_YEARS = 10
    BLOCKS_PER_YEAR = 525_600
    
    # Check if within distribution period
    blocks_since_genesis = block_number
    years_elapsed = blocks_since_genesis / BLOCKS_PER_YEAR
    
    if years_elapsed > 10:
        return 0  # Pool exhausted
    
    # Base bonus per block
    annual_distribution = CONSCIOUSNESS_POOL / DISTRIBUTION_YEARS
    bonus_per_block = annual_distribution / BLOCKS_PER_YEAR  # 1,569.63 ZION
    
    # Apply level multiplier
    MULTIPLIERS = {
        1: 1.0,   # PHYSICAL
        2: 1.05,  # EMOTIONAL
        3: 1.1,   # MENTAL
        4: 1.25,  # SACRED
        5: 1.5,   # QUANTUM
        6: 2.0,   # COSMIC
        7: 3.0,   # ENLIGHTENED
        8: 5.0,   # TRANSCENDENT
        9: 10.0   # ON_THE_STAR
    }
    
    multiplier = MULTIPLIERS.get(miner_level, 1.0)
    return bonus_per_block * multiplier
```

### PPLNS Payout System

The pool uses **Pay-Per-Last-N-Shares (PPLNS)** for fair distribution:

```python
def calculate_pplns_payout(block_reward, miner_shares, window_shares):
    """
    PPLNS distribution algorithm
    
    Args:
        block_reward: Total reward for the block
        miner_shares: Shares this miner contributed in window
        window_shares: Total shares in PPLNS window
    
    Returns:
        Miner's proportional payout
    """
    PPLNS_WINDOW = 100_000  # Last N shares
    POOL_FEE = 0.01  # 1%
    HUMANITARIAN_TITHE = 0.10  # 10%
    
    # Calculate net miner pool
    after_tithe = block_reward * (1 - HUMANITARIAN_TITHE)
    after_fee = after_tithe * (1 - POOL_FEE)
    
    # Proportional share
    miner_ratio = miner_shares / window_shares
    payout = after_fee * miner_ratio
    
    return payout
```
